# Laravel React CRUD (Latihan)

Project CRUD sederhana untuk latihan menggunakan Laravel + React (Inertia.js).

## Fitur

- Create Post
- Read Post
- Update Post
- Delete Post

## Teknologi

- Laravel
- React
- Inertia.js
- TypeScript
- MySQL

## Instalasi Singkat

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
```

## Jalankan

```bash
composer run dev
```

Akses: http://localhost:8000

## Catatan

Project ini dibuat khusus untuk latihan belajar CRUD, bukan untuk production.

## 1. Pendahuluan
Project ini merupakan implementasi aplikasi CRUD (Create, Read, Update, Delete) menggunakan Laravel sebagai backend dan React sebagai frontend dengan Inertia.js sebagai penghubung.

## 2. Teknologi yang Digunakan
- **Backend**: Laravel 12
- **Frontend**: React 19 + TypeScript
- **Bridge**: Inertia.js
- **Database**: MySQL
- **Styling**: Tailwind CSS

## 3. Fitur Aplikasi
- Menambah data post (Create)
- Menampilkan data post (Read)
- Mengubah data post (Update)
- Menghapus data post (Delete)

```bash
composer install
npm install
cp .env.example .env

php artisan migrate
```

